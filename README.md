# AeProp
AeProp is a full stack finantial dashboard designed for real state investors to track ROI, rental income and property maintenance

### Disclaimer
This is a personal/learning project to broaden my Angular knowledge, applying enterprise-grade patterns.
It is developed for educational and portfolio purposes to demostrate full-stack technical proeficiency using .NET Core, Angular 18 and Supabase stack.

* **Codebase:** 100% of the code in this repository was written from scratch. No proprietary code, private frameworks, or internal tools from any current or past employer were used

* **Logic:** The business logic implemented here is based on industry-standard CRUD operations and public domain real estate concepts

* **Data:** All data displayed (property listings, prices, images) is mock data or was manually created for testing purposes. No real-world customer databases or confidential company information were accessed or utilized
